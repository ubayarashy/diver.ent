1. OVERVIEW & OBJECTIVE
------------------------
GoSocial adalah digital marketing agency berbasis di Surabaya, Indonesia, yang
menyediakan layanan komprehensif mulai dari Social Media Management, Digital Ads,
SEO, Website Development, Branding, hingga Visual/Audio Production.

Tujuan website:
- Memperkenalkan GoSocial sebagai agensi terpercaya dan profesional
- Menghasilkan lead/konsultasi dari calon klien baru
- Membangun kepercayaan melalui portofolio, testimoni, dan trust signal
- Menyediakan portal klien aktif (Client Area)
- Meningkatkan konversi melalui multi-channel komunikasi

2. TARGET USERS
---------------
Primary  : Pemilik bisnis (UMKM, brand nasional, enterprise) yang mencari jasa
           digital marketing dan teknologi
Secondary: Marketing Manager / CMO perusahaan
Tertiary : Klien aktif yang mengakses Client Area

4. DESIGN REQUIREMENTS
-----------------------
Referensi: instrument.com
Aesthetic: Editorial, dark-mode bold, large typography, cinematic feel
Tone: Premium, modern, trustworthy, tech-forward

Palette:
- Background: #0a0a0a (near black)
- Surface: #111111
- Accent: #E8FF3A (electric lime/yellow-green)
- Text Primary: #F5F5F0
- Text Secondary: #888888
- Border: #222222

Typography:
- Display: Clash Display / Syne (bold, editorial)
- Body: DM Sans / Epilogue (clean, readable)

Key Visual Principles:
- Full-viewport hero sections
- Oversized typography with tight leading
- Asymmetric layouts, grid-breaking elements
- Smooth scroll animations (fade-up, stagger)
- Hover states yang memorable
- Generous whitespace di dark background

5. TECHNICAL REQUIREMENTS
--------------------------
- Single HTML file (all-in-one, no external build needed)
- Vanilla HTML + CSS + JavaScript (no framework dependency)
- Responsive: Mobile First (320px → 1440px+)
- Google Fonts via CDN
- CSS Variables untuk theming
- Smooth scroll behavior
- IntersectionObserver untuk scroll animations
- Accessible: ARIA labels, keyboard navigation
- Performance: lazy loading images, minimal JS
- Browser support: Chrome, Firefox, Safari, Edge (last 2 versions)

6. PAGES SCOPE (HOMEPAGE ONLY — FULL SINGLE PAGE)
---------------------------------------------------
Homepage mencakup semua section di atas sebagai one-page scrolling experience.
Sub-pages (About, Portfolio, Blog, Client Area, dll) direpresentasikan sebagai
anchor links / placeholder pages dalam file yang sama.

7. SUCCESS METRICS
-------------------
- Time on Page > 2 menit
- Bounce Rate < 50%
- CTA Click Rate (Konsultasi Gratis) > 5%
- WhatsApp click-through dari website
- Form submission rate

8. OUT OF SCOPE (v1.0)
-----------------------
- Backend server / database
- CMS integration
- E-commerce / payment gateway
- Client Area dashboard (placeholder only)
- Blog content management

9. TIMELINE (ESTIMATE)
-----------------------
Week 1: Design system, Hero, Navbar
Week 2: Core sections (Why Us, Services, Portfolio)
Week 3: Social proof, Footer, Modals
Week 4: QA, Performance audit, Launch
